# 4dot-machine-service

## ServiceNow Academy 2026 – Lab Exercises
### 📌 Overview

This project is part of the ServiceNow Academy 2026 program.

The goal is to design and build a custom ServiceNow application for 4Dot Machines, a company that delivers, maintains, and supports vending machines.

### 🏢 Business Context

4Dot Machines manages vending machines across multiple locations and requires a system to track machines, handle service requests, and resolve incidents efficiently.

### 🎯 Objectives

- Through this project, the application will support:

- Managing records of vending machines under service

- Handling refill requests for machines

- Processing orders for new vending machines

- Receiving and resolving incidents and issues

### ⚙️ Features (Planned)
- Custom tables for vending machines and services

- Service catalog items for refill and machine orders

- Incident management for machine issues

- Workflows and automation using ServiceNow

- Basic scripting (Business Rules, Client Scripts, UI Policies)

### 🛠️ Technologies
- ServiceNow Platform

- JavaScript (ServiceNow scripting)

- ITSM concepts

- Workflows & Flow Designer

### 🚧 Status

- Project in progress — built as part of hands-on lab exercises during the academy.

### 📚 Learning Focus
- Application development on ServiceNow

- Process automation and workflows

- Data modeling and platform architecture

- IT service management (ITSM) fundamentals
